# Samrt-Mirror-enabled-Home-Automation
The scope of this project includes designing a smart home automation system integrated with a smart mirror interface. The system focuses on real-time monitoring, centralized control, and efficient communication between IoT devices. It supports device control, environmental monitoring, and secure data transmission. The system can be extended to multiple rooms using distributed IoT nodes and can be accessed both locally and remotely through cloud connectivity.
